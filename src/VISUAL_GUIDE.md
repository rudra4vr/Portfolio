## ğŸ¬ Animations Overview

### On Page Load
```
1. Background particles fade in
2. Hero content slides up
3. Badge rotates and scales
4. Buttons fade in
5. Navigation slides up from bottom
6. Theme toggle rotates in
```

### On Scroll
```
1. Parallax background moves
2. Sections fade in as you scroll
3. Navigation highlights active section
4. Progress bar fills up
5. Timeline animates
```

### On Hover
```
1. Cards lift and rotate (3D effect)
2. Buttons scale and glow
3. Text changes color
4. Icons rotate
5. Borders glow
6. Cursor expands
```

### On Click
```
1. Navigation: Smooth scroll to section
2. Buttons: Scale down then up
3. Form: Loading spinner appears
4. Theme: Rotate and switch colors
```
